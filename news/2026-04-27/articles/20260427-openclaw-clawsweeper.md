# ClawSweeper: Open-source agent that autonomously triages, labels, and resolves GitHub issues using Claude

**Source**: The Decoder | **Published**: 2026-04-26 | **Topics**: AI Agents and Autonomous Systems

## Summary
ClawSweeper는 Claude API와 MCP(Model Context Protocol)를 활용해 GitHub 이슈를 자율적으로 분류(triage), 레이블링, 그리고 간단한 버그의 경우 직접 해결까지 수행하는 오픈소스 에이전트다. 출시 24시간 만에 GitHub Stars 3,200개를 돌파했다.

## Key Points
- Claude 3.7 Sonnet 기반, MCP를 통해 GitHub API·코드베이스 분석 툴 연동
- 이슈 분류 정확도 92%, 자동 레이블 부착률 88% 기록 (내부 테스트)
- 간단한 버그(타입 에러, 누락 변수 등) 자동 패치 PR 생성 기능 포함
- Apache 2.0 라이선스, 단독 서버 또는 GitHub Actions로 배포 가능
- 초기 24시간 내 Stars 3,200+, Forks 410+

## Why This Matters
MCP 생태계 기반의 실용적 에이전트 도구가 빠르게 확산되고 있음을 보여준다. 오픈소스 채택이 가속화되면 GitHub 이슈 관리의 표준 방식이 바뀔 수 있으며, AI 에이전트가 개발 운영(DevOps) 워크플로우에 깊숙이 통합되는 추세를 반영한다.
