# U.S. Commerce warns China is using model distillation to replicate frontier AI capabilities and bypass export controls

**Source**: MIT Technology Review | **Published**: 2026-04-25 | **Topics**: AI Policy, Regulation and Safety

## Summary
미국 상무부 산업안보국(BIS)이 발표한 보고서에 따르면, 중국 AI 연구기관들이 미국 프론티어 모델의 출력 데이터를 이용한 지식 증류(knowledge distillation) 기법으로 수출 통제 대상 칩 없이도 유사한 성능을 달성하고 있다는 증거를 확보했다. 보고서는 증류 훈련을 지원하는 클라우드 API 접근에 대한 추가 제한을 권고한다.

## Key Points
- BIS 보고서: 중국 기관 12곳 이상이 미국 모델 출력을 학습 데이터로 활용한 정황 포착
- 대상 모델: GPT-4급 이상 성능을 보이는 중국산 모델 다수
- 권고 조치: API 이용 시 국가·기관별 쿼리 패턴 모니터링 의무화 검토
- OpenAI·Anthropic·Google에 대해 비정상 대용량 API 호출 보고 의무 부과 예고
- 의회 AI 소위원회, 6월 청문회 일정 공지

## Why This Matters
칩 수출 통제가 무력화될 수 있다는 경고는 AI 지정학의 새로운 전선을 연다. API 접근 규제로 이어질 경우 오픈소스·클라우드 생태계 전반에 구조적 영향을 미친다.
