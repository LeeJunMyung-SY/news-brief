# 풀사이드, 온디바이스용 코딩 에이전트 모델 '라구나 XS.2' 오픈소스 출시

**Source**: AI타임스 | **Published**: 2026-04-29T17:23:14+00:00 | **Topics**: llm_models, ai_agents, ai_industry

## Summary

미국 AI 스타트업 풀사이드가 장시간 복잡 작업을 수행하는 에이전트 코딩 모델 ‘라구나(Laguna)’ 시리즈를 공개했다. 28일(현지시간) ‘라구나 M.1’과 ‘라구나 XS.2’를 첫 모델로 출시했으며, XS.2는 온디바이스용 오픈소스로 배포된다.

## Key Points

- ‘라구나 M.1’(서버용)·‘라구나 XS.2’(온디바이스 오픈소스) 동시 출시
- 정해진 도구를 호출하는 방식이 아니라 ‘스스로 코드를 생성·실행·조합’하는 에이전트형 학습
- 다중 작업의 병렬 처리, 새로운 시스템 구성까지 자율 수행 가능하도록 설계
- 온디바이스 코딩 에이전트 — 클라우드 의존 비용·지연·프라이버시 문제 동시 완화
- GitHub Copilot·Codex·Cursor 위주의 코딩 에이전트 시장에 새 카테고리 추가

## Why This Matters

코딩 에이전트는 LLM 비용 폭증의 핵심 원인이고(앤트로픽 클로드 코드 종량제 전환과 직결), 온디바이스 오픈소스 대안의 등장은 가격 압박을 가하는 첫 신호다.
