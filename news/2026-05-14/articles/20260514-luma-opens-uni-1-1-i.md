# Luma opens Uni-1.1 image model API at prices and quality matching OpenAI and Google

**Source**: The Decoder | **Published**: Wed, 13 May 2026 15:45:46 +0000 | **Topics**: llm_models

## Summary
Luma가 Uni-1.1 이미지 모델을 API로 공개하고 2,048픽셀 기준 이미지당 $0.04에 제공한다. Arena 리더보드에서 Google·OpenAI에 이어 3위이며 웹 검색·추론·이미지 편집까지 통합된 API라는 점이 차별점이다.

## Key Points
- 이미지당 $0.04 — 빅3 가격대와 동등
- Arena 리더보드 3위 — 품질 격차 거의 소멸
- 웹 검색·추론·편집 통합 API
- 생성 이미지 멀티모달 모델 시장 다극화

## Why This Matters
이미지 모델 가격·품질이 평준화되면 차별화는 워크플로우 통합으로 이동한다. 한국 SaaS·콘텐츠 기업의 이미지 AI 도입 비용은 더 떨어진다.
