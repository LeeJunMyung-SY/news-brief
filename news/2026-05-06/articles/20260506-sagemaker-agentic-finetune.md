# Amazon brings agentic fine-tuning to SageMaker with support for Llama, Qwen, Deepseek, and Nova
**Source**: The Decoder | **Published**: 2026-05-05T10:08:45+00:00 | **Topics**: ai_agents, llm_models

## Summary
아마존 웹서비스(AWS)가 SageMaker에 ‘에이전트형 파인튜닝’ 기능을 추가했다. 개발자가 모델 커스터마이즈를 직접 수행하는 대신 AI 에이전트가 데이터 준비·하이퍼파라미터 설정·평가까지 자동 수행한다. 라마·퀜·딥시크·노바를 기본 지원한다.

## Key Points
- AWS SageMaker에 ‘에이전트형 파인튜닝’ 기능 추가
- 에이전트가 데이터 준비·하이퍼파라미터·평가 자동 수행
- 라마·퀜(Qwen)·딥시크(DeepSeek)·노바(Nova) 즉시 지원
- 엔터프라이즈가 자체 모델 학습 진입장벽 추가 인하
- 오픈가중치 진영의 ‘훈련 자체가 에이전트가 되는’ 트렌드

## Why This Matters
파인튜닝의 가장 큰 비용은 ‘ML 엔지니어 인건비’였다. 에이전트가 이를 대체하면 산업·기업별 특화 모델이 폭발적으로 늘 수 있고, AWS는 모델 학습 시장의 표준 플랫폼 위치를 강화한다.
