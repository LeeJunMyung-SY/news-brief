# 메타, '더 오래 생각'하며 고품질 데이터 생성하는 ‘오토데이터’ 공개
**Source**: AI타임스 | **Published**: 2026-05-05T16:21:06+00:00 | **Topics**: llm_models

## Summary
메타가 AI 학습의 핵심 병목인 ‘데이터 품질’을 해결하기 위해 모델이 더 오래 사고하며 데이터를 생성·검증·개선하는 ‘오토데이터(Autodata)’ 프레임워크를 공개했다. 사람이 데이터를 큐레이션하던 단계에서 AI가 스스로 데이터 과학자처럼 반복 분석·개선을 수행하는 구조다.

## Key Points
- 메타 1일(현지시간) ‘오토데이터’ 프레임워크 공개
- AI가 데이터 생성→검증→개선 루프를 자체 수행
- 데이터 양 확장의 한계를 ‘심층 사고 시간’으로 보완
- 사람 큐레이션 의존도 감소, 사후학습 비용 절감 가능
- 고품질 합성 데이터 생성 표준 제안

## Why This Matters
공개 웹 텍스트 고갈 우려가 커지는 가운데 ‘더 오래 생각하는’ 합성 데이터 생성은 차세대 LLM 스케일링의 핵심 변수다. 메타가 자체 프레임워크를 공개한 것은 라마 후속 학습 방법론의 방향을 시사한다.
