# TSCG: Deterministic Tool-Schema Compilation for Agentic LLM Deployments

**Source**: arXiv CS.AI | **Published**: Thu, 07 May 2026 00:00:00 -0400 | **Topics**: ai_agents

## Summary
TSCG는 OpenAI Function Calling, Anthropic Tool Use, MCP가 전송하는 JSON 스키마를 LLM 친화 표현으로 결정론적으로 컴파일하는 어댑터로, 4B–14B 모델의 tool-use 실패 다수를 해소한다고 주장한다.

## Key Points
- JSON 스키마와 LLM 표현 간 mismatch 해소
- MCP·Function Calling·Tool Use 공통 적용
- small model 운영 비용 절감 가능

## Why This Matters
MCP 보급 흐름에서 tool 스키마 표준화·중개 레이어가 본격 산업화될 신호.
