# LCM: Lossless Context Management

**Source**: arXiv CS.AI | **Published**: Thu, 07 May 2026 00:00:00 -0400 | **Topics**: llm_models, ai_agents

## Summary
결정론적 LLM 메모리 아키텍처 LCM과 이를 적용한 코딩 에이전트 Volt를 제안한다. Opus 4.6 기반 OOLONG 벤치마크에서 32K~1M 토큰 전 구간에서 Claude Code보다 높은 점수를 기록했다고 주장한다.

## Key Points
- 결정론적 long-context 관리(LCM)로 손실 없는 컨텍스트 유지
- Opus 4.6 위에서 Claude Code 대비 OOLONG 점수 우위
- Recurs 계열 재귀 패러다임의 확장으로 자리매김

## Why This Matters
코딩 에이전트의 컨텍스트 한계가 실사용 신뢰도의 핵심 병목이라는 점에서, 결정론적 메모리 구조의 검증 사례로 의미가 있다.
