# RAG 시대 끝나나...'1200만 토큰' 컨텍스트 창 제공하는 모델 출시
**Source**: AI타임스 | **Published**: 2026-05-06T18:00:00+00:00 | **Topics**: llm_models

## Summary
미국 스타트업 서브쿼드래틱이 1200만 토큰 컨텍스트를 처리하는 신규 LLM ‘서브Q 1M-프리뷰’를 공개했다. 입력 길이에 따라 연산량이 선형적으로만 증가하는 구조로, 기존 트랜스포머 어텐션의 제곱 복잡도 한계를 우회했다고 회사 측은 밝혔다.

## Key Points
- ‘서브쿼드래틱’ 아키텍처 적용한 ‘서브Q 1M-프리뷰’ 공개
- 최대 컨텍스트 1200만 토큰 (책 수십 권 분량)
- 토큰 길이에 따른 연산량이 제곱이 아닌 선형으로 증가
- RAG 의존도 축소 가능 → 통째로 입력 후 추론하는 워크로드 부각
- 트랜스포머 SSM/하이브리드 계열 대안 아키텍처 가시화

## Why This Matters
1M~10M 토큰 영역은 실질적으로 RAG의 존재 이유를 줄인다. 길이가 길수록 비용이 폭발하던 추론 곡선이 완만해지면 장문서·코드베이스 분석 같은 워크로드의 경제성이 달라진다.
