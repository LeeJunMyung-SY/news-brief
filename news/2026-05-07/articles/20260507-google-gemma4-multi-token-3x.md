# Google speeds up Gemma 4 threefold with multi-token prediction
**Source**: The Decoder | **Published**: 2026-05-06T16:05:28+00:00 | **Topics**: llm_models

## Summary
구글이 Gemma 4 오픈 모델 패밀리에 멀티토큰 예측(multi-token prediction) 드래프터를 공개해 추론 속도를 최대 3배 끌어올렸다. 작은 보조 모델이 한 번에 여러 토큰을 제안하면 메인 모델이 한 번의 패스로 검증하는 방식이다.

## Key Points
- Gemma 4 오픈 가중치용 멀티토큰 예측 드래프터 공개
- 텍스트 생성 속도 최대 3배 향상
- 보조 모델 다중 후보 → 메인 모델 일괄 검증 구조
- 투기적 디코딩 계열 최적화의 정식 정착 신호
- 추론 비용·지연 직접 절감 → 온디바이스·엣지 활용 확대

## Why This Matters
추론 비용은 LLM 서비스의 손익을 결정한다. 메인 모델 변경 없이 3배 가속을 제공하는 드래프터는 오픈모델 진영의 가격 경쟁력을 한 번 더 키우는 변화다.
