# Anthropic taps SpaceX's Colossus-1 data center for 220,000 GPUs to power Claude
**Source**: The Decoder | **Published**: 2026-05-06T18:42:24+00:00 | **Topics**: ai_industry, llm_models

## Summary
앤트로픽이 SpaceX 산하 Colossus-1 데이터센터의 전체 컴퓨팅 용량(300MW 이상, 엔비디아 GPU 22만장)을 인수해 한 달 내 가동한다고 The Decoder가 보도했다. 이에 맞춰 클로드 코드와 클로드 API의 사용 한도도 두 배로 늘어난다.

## Key Points
- Colossus-1 데이터센터 전체 용량(300MW+, 엔비디아 GPU 약 22만장) 흡수
- 1개월 내 단계적 가동 예정
- Claude Code 및 Claude API 한도 2배 확대
- 구글 5GW 계약과 별도의 추가 컴퓨팅 라인
- 원래 xAI가 구축했던 인프라가 앤트로픽 손으로 이동한 첫 주요 사례

## Why This Matters
단일 모델 회사가 한꺼번에 22만장의 H급 GPU를 확보하는 시대가 됐다. 컴퓨팅이 이미 자체 자산이 아닌 '거래되는 상품'처럼 움직이고 있음을 보여준다.
