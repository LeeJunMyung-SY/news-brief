# GitHub Copilot switches to token-based billing in June 2026

**Source**: The Decoder | **Published**: 2026-04-28T09:19:53+00:00 | **Topics**: ai_industry, ai_agents

## Summary

마이크로소프트 GitHub가 Copilot 요금제를 6월 16일부터 정액(월 $10/$19) 모델에서 사용량/토큰 기반 모델로 전환한다고 발표했다. 에이전트 모드와 신규 멀티파일 작업이 토큰 소비를 빠르게 늘리는 상황에서 비용 책임을 사용자에게 이전하는 구조다.

## Key Points

- GitHub Copilot 6월 16일자로 토큰 기반 청구 전환
- Free 플랜은 월 50회 채팅·2000회 자동완성 등 한도 명문화
- Pro/Business는 월별 토큰 한도 + 초과분 종량제
- Codex/Claude/GPT 등 모델별 가중치 차등(고급 모델 토큰 가중)
- '에이전트 모드'에서 토큰 소비가 폭증한 데 대한 직접 대응

## Why This Matters

코딩 AI 시장에서 가장 큰 단일 제품의 가격 모델 변경은, 경쟁사(Cursor·Cody·Codeium)와 사내 도구의 비용 비교 기준점을 일제히 재설정하며, '에이전트 토큰 가격'을 기업 IT 의사결정의 일상적 변수로 만든다.
