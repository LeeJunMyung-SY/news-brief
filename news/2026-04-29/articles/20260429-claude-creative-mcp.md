# Claude can now plug directly into Photoshop, Blender, and Ableton

**Source**: The Verge AI | **Published**: 2026-04-28T12:49:08-04:00 | **Topics**: ai_agents

## Summary

Anthropic이 Claude Desktop을 Adobe Photoshop, Blender, Ableton Live 세 도구에 직접 연결하는 공식 MCP 커넥터를 공개했다. Claude가 레이어·3D 노드·DAW 트랙을 파일 시스템처럼 읽고 직접 명령을 실행하며, 콘텐츠 제작 워크플로 전반에서 에이전트가 행위자로 작동한다.

## Key Points

- Photoshop: 레이어·마스크·필터를 Claude가 직접 조작
- Blender: 3D 씬 그래프 읽기/수정 + 렌더 작업 큐잉
- Ableton Live: MIDI 트랙·오디오 클립 자동 편집
- 기반은 모두 Anthropic의 공식 MCP 커넥터 — 서드파티 의존 X
- Claude Desktop의 'Computer Use'와 별개의 도구별 직접 통합 경로

## Why This Matters

에이전트가 OS 단의 'Computer Use'를 넘어 도메인 특화 도구의 내부 객체 모델까지 직접 조작하는 단계로 진입했다는 신호로, 콘텐츠·디자인·미디어 제작 전반에서 에이전트가 협업자에서 실행 주체로 이동한다.
