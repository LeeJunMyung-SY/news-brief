# Amazon is already offering new OpenAI products on AWS

**Source**: TechCrunch AI | **Published**: 2026-04-28T19:48:02+00:00 | **Topics**: ai_industry

## Summary

MS 독점 계약 종료 발표 다음 날, AWS Bedrock에서 OpenAI 신규 모델이 곧바로 제공되기 시작했다. AWS는 OpenAI의 새 GPT 계열 모델과 코드 모델을 자체 인프라에 호스팅하며, OpenAI는 마이크로소프트 외 클라우드에서 처음으로 자사 모델을 직접 판매할 수 있게 됐다.

## Key Points

- OpenAI–MS 계약 개편 직후 24시간 내 AWS Bedrock 카탈로그에 OpenAI 모델 등재
- AWS는 자체 데이터센터에서 OpenAI 가중치를 호스팅하는 첫 외부 클라우드
- 양사 모두 가격·SLA·지역 가용성 세부 사항을 곧바로 공개
- AWS는 별도 보도자료에서 'OpenAI를 즉시 사용할 수 있다'고 강조
- Anthropic도 같은 카탈로그에 있는 만큼 AWS는 빅3 프런티어 모델을 모두 호스팅

## Why This Matters

MS 독점이라는 OpenAI의 구조적 제약이 단순한 법적 문구 변경에 그치지 않고 사업적으로 실재하는 멀티클라우드 공급으로 즉시 전환됐다는 점에서, OpenAI의 IPO 경로와 AWS의 프런티어 모델 카탈로그 전략 모두에 직접적인 변곡점이다.
