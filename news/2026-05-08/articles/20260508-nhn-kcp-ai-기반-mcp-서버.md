# NHN KCP, AI 기반 MCP 서버 도입…"명령만으로 결제 연동 코드 생성"

**Source**: ZDNet Korea | **Published**: Fri, 08 May 2026 10:10:06 +0900 | **Topics**: ai_agents, ai_industry

## Summary
NHN KCP가 AI 모델·외부 시스템 통신 표준 MCP를 도입했다. 자연어 명령만으로 결제 연동 코드를 자동 생성할 수 있고, 구글 클라우드의 AI 에이전트 결제 프로토콜 ‘AP2’에도 초기부터 참여 중이다.

## Key Points
- MCP 기반 자연어 → 결제 연동 코드 자동 생성
- 기존 API 문서 기반 수작업 코딩 대체
- 구글 AP2 AI 에이전트 결제 프로토콜에 한국사 초기 참여
- 에이전트가 사용자 대신 구매·결제 수행 시나리오 대비

## Why This Matters
AI 에이전트가 직접 결제를 수행하는 ‘에이전트 페이먼츠’ 시대를 대비한 한국 결제 인프라 적응 사례로, 향후 결제 표준·소비자 보호 논의에 영향을 줄 수 있다.
