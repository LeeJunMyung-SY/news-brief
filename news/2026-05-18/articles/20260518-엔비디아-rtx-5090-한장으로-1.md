# 엔비디아, 'RTX 5090' 한장으로 1분 영상 생성하는 월드 모델 오픈 공개

**Source**: AI타임스 | **Published**: 2026-05-17 18:01:12 | **Topics**: llm_models, ai_industry, physical_ai_robotics

## Summary
엔비디아가 단일 RTX 5090 GPU로 1분 720p 영상을 생성하는 26억 파라미터 월드 모델 SANA-WM을 아카이브로 공개했다. 카메라 움직임·행동 정보를 입력받아 영상 시퀀스를 만든다.

## Key Points
- RTX 5090 단일 GPU에서 1분 720p 생성
- 26억 파라미터, 비교적 경량
- 월드 모델 효율성 도약 — 기존엔 대규모 GPU 필요
- 로봇 훈련·시뮬레이션 진입 장벽 낮춤
- 오픈 공개

## Why This Matters
고가 GPU 클러스터 없이 월드 모델 연구가 가능해진다. 한국 로봇·자율주행 스타트업의 R&D 비용 구조에 직접 영향.
