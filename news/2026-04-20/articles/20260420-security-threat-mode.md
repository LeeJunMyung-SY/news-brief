# Security Threat Modeling for Emerging AI-Agent Protocols: A Comparative Analysis of MCP, A2A, Agora, and ANP

**Source**: arXiv CS.AI | **Published**: Mon, 20 Apr 2026 | **Topics**: AI Agents and Autonomous Systems

## Summary
Model Context Protocol(MCP), Agent-to-Agent(A2A), Agora, Agent Network Protocol(ANP) 등 새롭게 등장한 AI 에이전트 통신 프로토콜들의 보안 위협을 비교 분석한다. 각 프로토콜의 공격 표면과 취약점을 체계적으로 분류하고, 에이전트 간 통신 보안 강화를 위한 권고 사항을 제시한다. 에이전트 생태계가 급성장하는 시점에 보안 설계의 기초를 마련한다.

## Key Points
- MCP, A2A, Agora, ANP 4가지 에이전트 프로토콜의 보안 위협 체계적 비교
- 각 프로토콜별 고유한 공격 표면과 취약점 식별
- 에이전트 통신 보안 강화를 위한 설계 원칙 제안
- 에이전트 프로토콜 표준화 논의에 보안 관점 통합 필요성 강조

## Why This Matters
MCP가 광범위하게 채택되고 에이전트 간 통신 프로토콜이 표준화되는 시점에, 이들의 보안 위협을 사전에 파악하고 대응하는 것은 전체 에이전트 생태계의 안전성을 결정한다.
