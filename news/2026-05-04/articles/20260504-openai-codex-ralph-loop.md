# 오픈AI 버전 '랄프 루프' 등장...코덱스 CLI에 '자율 목표' 기능 추가
**Source**: AI타임스 | **Published**: 2026-05-02T19:14:18+09:00 | **Topics**: ai_agents, llm_models

## Summary
오픈AI가 1일(현지시간) 코덱스 CLI 0.128.0 버전에 장기 목표를 자율 수행하는 '/goal' 기능을 추가했다. 사람이 중간에 개입하지 않아도 코덱스가 며칠에 걸쳐 작업을 이어가는 방식으로, 일명 '랄프 루프(Ralph loop)'에 해당한다. 개발자가 목표만 설정하면 달성될 때까지 AI가 스스로 코딩을 이어간다.

## Key Points
- 코덱스 CLI 0.128.0에 /goal 기능 추가
- 사람 개입 없이 며칠 단위 자율 실행
- 이른바 '랄프 루프(Ralph loop)' 방식
- 목표 설정만으로 장기 코딩 작업 자동화
- OpenAI의 첫 정식 장기 자율 에이전트 인터페이스

## Why This Matters
자율 코딩 에이전트가 며칠 단위 장기 작업으로 확대되며 개발자 워크플로의 자동화 폭이 크게 넓어짐.
