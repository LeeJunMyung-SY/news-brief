# 카카오, '플레이MCP'에 오픈클로 연동 지원..."나만의 AI 비서 직접 만든다"
**Source**: AI타임스 | **Published**: 2026-05-01T17:20:29+09:00 | **Topics**: ai_agents, ai_industry

## Summary
카카오가 모델 컨텍스트 프로토콜(MCP) 기반 개방형 플랫폼 '플레이MCP'에 오픈소스 AI 에이전트 '오픈클로'를 연동 지원한다고 1일 밝혔다. 플레이MCP에는 카카오톡 톡캘린더·카카오맵·선물하기·멜론 등 카카오 서비스를 포함해 약 200여개 외부 MCP 서버가 등록돼 있어, 사용자가 로컬 PC에 설치한 자체 LLM을 다양한 도구·서비스에 자유롭게 붙여 쓸 수 있게 됐다.

## Key Points
- 플레이MCP에 오픈클로 연동
- 카카오 자사 서비스 포함 200+ MCP 서버 등록
- 사용자가 로컬에 LLM·에이전트 설치
- 원하는 LLM을 외부 도구에 자유 결합
- 한국 빅테크 첫 본격 MCP 생태계 참여

## Why This Matters
한국 빅테크가 MCP 생태계에 적극 참여하면서 일반 사용자용 AI 에이전트 환경의 진입 장벽이 낮아짐.
