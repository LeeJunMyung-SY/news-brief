# This startup's new mechanistic interpretability tool lets you debug LLMs
**Source**: MIT Technology Review | **Published**: 2026-04-30T15:59:41+00:00 | **Topics**: llm_models, ai_policy

## Summary
샌프란시스코 스타트업 Goodfire가 학습 중 AI 모델 내부를 들여다보고 파라미터를 직접 조정할 수 있는 mechanistic interpretability 도구 'Silico'를 공개했다. 모델 제작자가 학습 도중에 모델 행동을 정밀하게 제어·디버그할 수 있게 되며, 정렬·해석 가능성 연구의 상용 단계 진입을 알리는 사례다.

## Key Points
- 학습 중 LLM 내부 직접 관찰
- 파라미터 단위 조정으로 행동 제어
- mechanistic interpretability 상용화
- 디버깅·정렬·해석 연구에 활용
- Goodfire 'Silico' 출시

## Why This Matters
AI 모델의 black-box 문제를 푸는 도구가 상용 제품 단계에 진입한 첫 사례로, 안전·정렬 연구의 인프라화 신호.
