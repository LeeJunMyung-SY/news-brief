# Running Codex safely at OpenAI

**Source**: OpenAI Blog | **Published**: Fri, 08 May 2026 12:30:00 GMT | **Topics**: ai_agents, ai_policy

## Summary
OpenAI가 Codex 코딩 에이전트를 사내에서 안전하게 운영하기 위해 적용한 보안 모델을 공개했다. 샌드박싱, 승인 흐름, 네트워크 정책, 에이전트 전용 텔레메트리 등을 결합해 컴플라이언스를 충족하면서도 코딩 에이전트 채택을 확대할 수 있도록 설계했다.

## Key Points
- 샌드박싱과 자동 승인 정책으로 위험한 명령을 차단
- 네트워크 정책으로 에이전트가 접근 가능한 외부 자원 제한
- 에이전트 행동을 추적하는 전용 텔레메트리 도입
- 조직 보안 표준에 부합하는 코딩 에이전트 운영 가이드 제공

## Why This Matters
코딩 에이전트의 기업 도입 확산기에 가장 큰 걸림돌인 보안·거버넌스 모델을 OpenAI가 공식적으로 정리했다. 사내 적용 사례라는 점에서 다른 기업의 도입 청사진으로 직접 활용 가능하다.
