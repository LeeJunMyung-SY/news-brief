# 앤트로픽, ‘클로드 코드’에 멀티 에이전트 관리 기능 추가

**Source**: AI타임스 | **Published**: 2026-05-12 18:22:21 | **Topics**: ai_agents, ai_industry

## Summary
Anthropic이 Claude Code에 다수 AI 에이전트를 한 화면에서 관리·전환하는 'Agent View' 연구 프리뷰를 공개했다. CLI 환경에서 실행되는 여러 Claude Code 세션을 단일 인터페이스로 통합 운영해 사실상 AI 코딩 도구를 '에이전트 운영체제' 수준으로 확장한다.

## Key Points
- CLI 기반 다중 Claude Code 세션을 한 화면에서 시각화·전환·승인
- 단일 에이전트 코딩에서 'fleet 관리' 수준으로 추상화 레벨 상승
- OpenAI Codex CLI, Cursor 등 경쟁 도구의 UX 기준선을 다시 설정
- 엔터프라이즈 코딩에서 'human-in-the-loop' 운영 모델 표준화 가능성

## Why This Matters
AI 코딩의 단위가 '한 명의 에이전트'에서 '에이전트 팀'으로 옮겨간다. 사내 개발 조직의 인력 운영과 권한·승인 워크플로우 설계에 직결.
